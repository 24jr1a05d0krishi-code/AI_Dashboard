# 📊 AI Dashboard - IPL Data Analysis

## 📌 Project Overview
This project is an interactive AI Dashboard built using Streamlit to analyze IPL match data.

---

## 🚀 Features
- Dataset Overview
- KPIs (Key Metrics)
- Data Cleaning
- Visualizations (Bar, Pie, Histogram)
- Interactive Filters

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

---

## 📂 Dataset
IPL Matches 2008–2022

---

## ▶️ How to Run
```bash
pip install -r requirements.txt
python -m streamlit run app.py
